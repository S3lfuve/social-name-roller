## Зачем нужна программа и что она делает

**TG Names Roller** помогает искать редкие, интересные и потенциально свободные username для разных площадок.

Программа подходит для тех, кто хочет:
- подбирать необычные никнеймы под Telegram, Fragment и другие сервисы
- генерировать собственные списки кандидатов по гибким правилам
- быстро отсеивать занятые варианты
- искать короткие, редкие или стилистически подходящие username под конкретную задачу

Поддерживаемые площадки проверки:
- Telegram
- Fragment
- Instagram
- X
- TikTok
- YouTube
- GitHub

## Принцип работы программы

Работа строится в два этапа:

1. **Генерация кандидатов**
   - программа собирает или создает список username по заданным правилам
   - можно генерировать варианты из английских слов, транслита русских слов или случайных букв
   - можно управлять длиной, редкостью слов, количеством слов в нике, префиксом и суффиксом

2. **Проверка списка**
   - программа берет готовый список username и последовательно проверяет его по выбранным площадкам
   - занятые варианты отбрасываются
   - свободные варианты можно сохранять отдельно
   - порядок проверки можно менять: по порядку, случайно, от коротких к длинным или наоборот

Для обычного пользователя этого достаточно: программа помогает сначала собрать подходящие варианты, а потом быстро отделить перспективные username от занятых.

## Основные возможности

### Генератор username

- генерация username из английских слов
- генерация username через транслит русских слов
- генерация случайных строк из латинских букв
- настройка общей длины username
- настройка количества слов в одном нике
- выбор качества слов: от более частых до более редких
- добавление собственного префикса в начало
- добавление собственного суффикса в конец
- опциональная вставка артиклей `a`, `an`, `the`
- удаление повторяющихся букв внутри отдельных слов
- использование `seed` для повторяемой генерации
- настройка глубины поиска слов через параметр `scan`

### Проверка username

- проверка списка никнеймов по нескольким площадкам
- выбор нужных сервисов перед запуском
- параллельная проверка с настройкой количества workers
- ограничение числа проверяемых строк
- настройка задержки между проверками
- экспорт найденных FREE username в `found.txt`
- режим скрытия занятых вариантов `BUSY`
- поддержка паузы и корректной остановки проверки

### Интерфейс

- графический запуск через `run_gui.bat`
- консольный запуск для ручной настройки через CLI
- поддержка **русского и английского языка интерфейса**
- сохранение и загрузка конфигов
- автосохранение последних настроек

## Доступные фильтры и настройки поиска

Программа позволяет гибко собирать желаемый никнейм:

- **Длина username**
  - точная длина
  - минимальная и максимальная длина

- **Редкость слова**
  - диапазон качества от более популярных слов к более редким

- **Количество слов в нике**
  - одинарные username
  - составные username из нескольких слов

- **Источник генерации**
  - английские слова
  - транслит русских слов
  - случайные латинские буквы

- **Форма username**
  - собственный префикс
  - собственный суффикс
  - артикли
  - схлопывание повторяющихся букв

- **Фильтры проверки**
  - выбор площадок
  - лимит строк
  - порядок проверки
  - количество workers
  - задержка между запросами

## Способ запуска

### Вариант 1. Запуск через GUI

1. Откройте папку проекта.
2. При первом запуске установите зависимости через `install_deps.bat`.
3. После этого запустите `run_gui.bat`.

Также можно запускать GUI напрямую через `TG Names Roller.pyw`.

### Вариант 2. Запуск через CLI

Сначала установите зависимости:

```bat
install_deps.bat
```

Далее можно использовать консольные команды для генерации и проверки.

## Консольная команда для установки и запуска программы

### Установка зависимостей

```bat
py -3 -m venv .venv
.venv\Scripts\python.exe -m pip install -r requirements.txt
```

### Запуск GUI

```bat
run_gui.bat
```

### Пример генерации username

```bat
.venv\Scripts\python.exe generate_words.py --out words.txt --count 300 --min-len 5 --max-len 10 --min-quality 3 --max-quality 5 --min-words 1 --max-words 2 --generator-mode translit --prefix neo --suffix x
```

### Пример проверки username

```bat
.venv\Scripts\python.exe tg_fragment_username_checker.py --wordlist words.txt --check-telegram --check-fragment --check-instagram --check-x --check-tiktok --check-youtube --check-github --workers 4 --delay 0.2 --order random --found-out found.txt
```

## Краткое описание файлов

### Основные файлы

- `tg_names_roller_gui.py`
  - основной графический интерфейс программы
  - объединяет генератор, проверку, конфиги и настройки интерфейса

- `generate_words.py`
  - консольный генератор username
  - создает списки никнеймов по заданным параметрам: длина, качество, режим генерации, префикс, суффикс и другое

- `tg_fragment_username_checker.py`
  - консольный модуль проверки username
  - проверяет список никнеймов по выбранным площадкам и выводит статусы `FREE`, `BUSY`, `UNKNOWN`

- `TG Names Roller.pyw`
  - легкий launcher для запуска GUI без ручного открытия основного `.py` файла

### Вспомогательные файлы

- `run_gui.bat`
  - быстрый запуск графической версии на Windows

- `install_deps.bat`
  - установка зависимостей в локальное виртуальное окружение `.venv`

- `requirements.txt`
  - список Python-зависимостей проекта

### Папки и данные

- `configs/`
  - сохраненные конфиги и последние настройки интерфейса

- `assets/`
  - графические ресурсы программы

- `words.txt`
  - список сгенерированных или подготовленных username для дальнейшей проверки

- `found.txt`
  - файл с найденными FREE username

## Для кого подойдет проект

Проект может быть полезен:
- реселлерам и исследователям username-ниш
- пользователям, которые подбирают редкий ник для личного бренда

## Примечания

- некоторые платформы могут отвечать нестабильно или ограничивать частые запросы
- часть username требует ручной проверки перед использованием
- чем выше `workers` и ниже `delay`, тем выше шанс упереться в лимиты
